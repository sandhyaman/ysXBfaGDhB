# Python企业员工管理系统

## 核心功能
- 部门管理
- 用户管理
- 订单管理
- 任务管理
- 手机管理
- 设备管理
- 管理员管理
- 订单数据可视化
- 修改个人信息

## 适用场景
- 企业内部员工信息管理
- 企业订单处理与跟踪
- 任务分配与进度监控
- 手机与设备信息维护
- 数据分析与可视化展示

## 技术特点
- 后端：Python、Django
- 前端：HTML、CSS、JavaScript、jQuery
- 数据库：MySQL
- 开发工具：Vscode、Navicat
- 开发环境：Python 3.0以上、MySQL 8.0
